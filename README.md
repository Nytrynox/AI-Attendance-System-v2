# AI Attendance System v2.0

Cloud-synced facial recognition attendance system with live anti-spoofing detection and mobile companion app.

## Features

- Cloud-synced attendance records via MongoDB Atlas
- Advanced anti-spoofing with 3D depth estimation
- Mobile app for student self-check-in
- Real-time attendance dashboard
- Multi-camera support for large venues
- Automated absence notifications

## Tech Stack

Python, OpenCV, MongoDB Atlas, Flutter, Flask, TensorFlow

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Attendance-System-v2.git
cd AI-Attendance-System-v2
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```
